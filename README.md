### AI-Powered Virtual Interview Practice Platform

AI Mock Interviewer is a **full-stack intelligent web application** designed to help students and professionals **practice mock interviews** in a realistic, interactive, and AI-driven environment.  
The platform simulates real interview scenarios, generates intelligent interview questions, evaluates user responses, and provides **AI-generated feedback** to improve communication, technical accuracy, and confidence.

It integrates **Gemini API** for question generation and AI evaluation, and **Clerk Authentication** for secure user access, privacy, and identity management.
